The 
SentSpace
hosted interface is retired as of 2024 due to technical challenges. However, it's easy to run it locally! Head over to the github repository to set it up! Note that in addition to lexical features, if you'd like to use contextual/syntactic features, you'd have to additionally set up and run the PCFG syntax server and provide the 
host:port
as an argument to 
SentSpace
.
